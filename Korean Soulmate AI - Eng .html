Creation Process

  1. Starting from pain points: Interviewed Hong Kong K-fans + reflected on own struggles → identified "loneliness + output anxiety + motivation loss" as core issues
  2. Rapid iterative design: Sketched wireframes in Figma → wrote detailed Feature Design doc → built interactive prototype with CodePen + HTML/CSS/JS (completed in 3–4 days)
  3. Tool choices & reasons:
    • Doubao AI → fast generation of high-quality, cute Korean-style mockups
    • CodePen → instant preview & easy sharing (perfect for course demo, no installation needed)
    • HTML (Grok / Perplexity) → created truly clickable interactive prototype (demonstrates real web skills)
